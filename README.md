Bug reports, feature requests, and documentation for my [Timberborn](https://timberborn.com/) mods.

The mod source lives in private repositories — **this repo is the public home for issues and docs.**
Found a bug or have an idea? **[Open an issue →](https://github.com/jamesjacko/timberborn-mods-hub/issues/new/choose)**

## Report a bug or request a feature

**[Open an issue →](https://github.com/jamesjacko/timberborn-mods-hub/issues/new/choose)**

For bugs, please include your Timberborn version, the steps to reproduce, and your `Player.log`
if you can grab it:

- **macOS:** `~/Library/Logs/Mechanistry/Timberborn/Player.log`
- **Windows:** `%USERPROFILE%\AppData\LocalLow\Mechanistry\Timberborn\Player.log`
