# Grid Cutting Patterns

Mark trees for cutting in a **grid pattern** instead of clear-cutting. The trees left standing
reseed the gaps, so a forest keeps regrowing while you harvest it — ideal in the early game,
before the Forester is unlocked.

**▶ [Get it on the Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3739849811)**

## How to use

The tools live **inside the game's own tree-cutting menu**, next to the standard cut/uncut tools.
Pick a pattern, drag-select an area, and only the tiles in that pattern get marked.

- The tile you **start the drag on is always marked** for cutting, and the pattern stays anchored
  there as you drag in any direction.
- Re-applying a pattern over an area **replaces** the marks rather than stacking them.
- It reuses the game's own selection, highlight and the green "already marked for cutting" overlay,
  so it feels native.

## Patterns

| Pattern | Cuts | Leaves standing |
|---------|------|-----------------|
| **Checkerboard** | every other tree (~50%) | a checkerboard of seed trees |
| **Horizontal Stripes** | alternate rows (~50%) | full horizontal seed rows |
| **Vertical Stripes** | alternate columns (~50%) | full vertical seed rows |
| **Thin** | one tree in four (~25%) | the most trees — fastest regrowth |
| **Heavy** | three trees in four (~75%) | a sparse seed grid — the most wood |
| **Sparse** | all but a widely-spaced 1-in-5 grid (~80%) | each seed tree ringed by gaps — the most efficient reseeding |

The **Sparse** pattern was suggested by a player (yo.bo): a tile's chance of a new tree or bush
spawning doesn't rise with the number of trees around it, so a maximally-spread seed grid gives
the most efficient natural reseeding.
