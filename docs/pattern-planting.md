# Pattern Planting

Plant two crops in a repeating **pattern** with a single drag — no more placing fields one strip at
a time.

> **Coming soon.** This mod isn't on the Steam Workshop yet. This page will get a Workshop link
> when it's released.

## How to use

The tool lives in its own group on the **bottom bar**. Pick any two crops and a layout, then
drag-select the area you want planted. The crop list is filtered to your faction, so you only see
crops you can actually plant.

## Patterns

| Pattern | Layout |
|---------|--------|
| **Checkerboard** | the two crops alternate every tile |
| **Horizontal rows** | crop A and crop B in alternating rows |
| **Vertical columns** | crop A and crop B in alternating columns |

Every tile is planted only where that crop can **actually grow**, so the pattern fits the terrain
instead of fighting it.
